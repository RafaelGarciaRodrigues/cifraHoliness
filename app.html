      Manual do Aplicativo de Cifras
      
  
  
      📱 Navegação Geral
      
  
        
    
      • Desktop: Use as teclas →/← ou ↑/↓ ou PageUp/PageDown para navegar entre acordes
        • 
    
      • Mobile: Toque no botão circular azul (canto inferior) para avançar acordes
        • 
    
      • Toque longo (2s): Volta ao primeiro acorde da primeira música
        • 
    
      • Tecla M: Mostra/oculta menu lateral esquerdo
        • 
    
      • Tecla F: Mostra/oculta braço do violão
        • 
    
      • Tecla D: Abre "Doxologia" em tela cheia
        • 
  
      

  
      🎵 Gerenciamento de Músicas
      
  
        
    
      • Lista de músicas: Clique simples para selecionar música
        • 
    
      • Fila de execução: Ctrl+clique (Cmd+clique no Mac) para adicionar/remover da fila
        • 
    
      • Filtro por categoria: Use o seletor de categoria para filtrar músicas por tipo
        • 
    
      • Transposição: Cada música pode ser transposta individualmente - use o seletor de tom
        • 
  
      

  
      🎸 Braço do Violão
      
  
        
    
      • Visualização: Mostra escala (azul), acordes (verde) e baixo (anel laranja)
        • 
    
      • Capo: Ajuste a posição do capo para transpor fisicamente
        • 
    
      • Som: Clique nas bolinhas coloridas para ouvir as notas
        • 
    
      • Cordas soltas: Clique nas cordas (fundo) para ouvir cordas soltas
        • 
  
      

  
      🎼 Acordes e Cifras
      
  
        
    
      • Navegação: Clique diretamente nos acordes ou use as setas do teclado
        • 
    
      • Destaque automático: Linha de letra correspondente fica destacada em amarelo
        • 
    
      • Som automático: Acordes tocam automaticamente quando selecionados (se habilitado)
        • 
    
      • Marcadores de dinâmica: Linhas com traços (---, ----, etc) controlam volume
        • 
  
      

  
      🎛️ Controles de Áudio
      
  
        
    
      • Timbre: Escolha entre Strings, Voz Humana, Pad Atmosférico ou Bass Elétrico
        • 
    
      • Envelope: Ajuste Attack (tempo de entrada) e Release (tempo de saída)
        • 
    
      • Chorus: Configure Wet (intensidade), Rate (velocidade), Depth (profundidade), Width (abertura estéreo)
        • 
    
      • Reverb: Ajuste Wet (intensidade), Seconds (tamanho da sala), Decay (decaimento)
        • 
    
      • Timbre: Brilho (corta frequências altas) e Ar (realce de agudos)
        • 
  
      

  
      ⚙️ Configurações
      
  
        
    
      • Fonte: Ajuste o tamanho da fonte das cifras (15-75px)
        • 
    
      • Reset: Botão vermelho restaura configurações de áudio para padrão
        • 
    
      • Persistência: Todas as configurações são salvas automaticamente no navegador
        • 
  
      

  
      🖨️ Impressão
      
  
        
    
      • Ctrl+I: Modo impressão de cifras - imprime todas as músicas da fila em 2 colunas
        • 
    
      • Ctrl+L: Modo impressão de lista - imprime título, tom e YouTube de músicas da categoria atual
        • 
    
      • ESC: Fecha o modo de impressão
        • 
    
      • Dica: Use Ctrl+P (Cmd+P no Mac) após abrir o modo de impressão desejado
        • 
  
      

  
      📖 Recursos Especiais
      
  
        
    
      • Versículos bíblicos: Aparecem aleatoriamente no topo das músicas (categoria Holiness)
        • 
    
      • Auto-scroll: Centraliza automaticamente o acorde ativo na tela
        • 
    
      • Sequência automática: Ao chegar no último acorde, avança para próxima música da fila
        • 
    
      • Volume dinâmico: Marcadores de dinâmica (---, ----) ajustam volume automaticamente
        • 
    
      • Tela bíblica: Após última música da fila, exibe versículos rotativos em tela cheia
        • 
  
      

  
      🔧 Dicas Técnicas
      
  
        
    
      • Formato de acordes: Suporta acordes complexos como Em7(9), A4(7/9), D/F#, etc.
        • 
    
      • Transposição inteligente: Preserva inversões e extensões ao transpor
        • 
    
      • Áudio Web: Usa Web Audio API para síntese em tempo real (requer navegador moderno)
        • 
    
      • Responsivo: Interface adapta automaticamente para mobile/desktop
        • 
  
      

  
      ⌨️ Atalhos do Teclado
      
  
        
    
      • → / ↓ / PageDown: Próximo acorde
        • 
    
      • ← / ↑ / PageUp: Acorde anterior
        • 
    
      • M: Mostra/oculta menu lateral
        • 
    
      • F: Mostra/oculta braço do violão
        • 
    
      • D: Abre Doxologia em tela cheia
        • 
    
      • Ctrl+I: Modo impressão de cifras
        • 
    
      • Ctrl+L: Modo impressão de lista
        • 
    
      • ESC: Fecha modo impressão ou ajuda
        • 
  
      

  
      
    💡 Dica: Para melhor experiência, use fones de ouvido e permita que o navegador reproduza áudio automaticamente.
